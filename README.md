# task1_landing_page
Responsive Landing Page

A simple and fully responsive Landing Page built using HTML5 and CSS3.
This project demonstrates modern web layout techniques including Flexbox and media queries, along with a clean header, hero section, and footer.


 Objective

To build a clean, responsive landing page that includes:

A header with logo and navigation links

A hero section with heading, description, and a call-to-action button

A footer with social media links

Mobile-friendly layout using CSS media queries
